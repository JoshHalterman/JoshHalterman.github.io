Description

This is a simulator designed to allow you to see how a Turing machine works. You can write your own instructions, or use one of the preset buttons to automatically load a set of instructions.

For further details, follow this link:
https://en.wikipedia.org/wiki/Turing_machine

Once you've created your instructions, click the Load button to load your instructions into the machine. Then, click the Step button to step through your instructions one by one.

How to Write Instructions

This Turning machine simulator supports 6 instructions, based on the machine state and current symbol.

Each instruction consists of 3 parts: the symbol that will be written (either a 0 or a 1), the direction the tape will move (either Left or Right), and the new state the machine will change to. (either state A, state B, state C, or the Halt state, which ends the program)
